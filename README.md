## How to install
> npm install d-math --save

## How to use use
```javascript
    import dMath from 'd-math'
    let result = dMath.add(1,9,3,5,7);
```

```javascript
    import add from 'd-math/lib/add'
    let result = add(1,9,3,5,7);
```
